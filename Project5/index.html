_VOIS New Joiner Document

Steps to be followed

  1. Request to VDI Setup.
  2. Get Access to Repository
  3. Request to Jump Server Access.
  4. Connect to RDP using Jump Server
  5. Install the required Software for Development.
  6. Reference IP for VDI setup
  7. VPN setup along with MFA (Multi Factor Authentication)

1.Request to VDI Setup.

Process for VDI creation and access –

  1. Request to set up a new VDI is done by raising a ticket in Jira. Jira board is accessible when access is granted.. Concerned person Tushar Singh Raghuvanshi Vodafone.
  2. Link to Jira board PD board - Agile Board - CPS Jira (vodafone.com)
  3. Someone from the PaaS Team will pick the ticket and start creating VDI for your request.
  4. Provide team members VDI IP for setup configuration reference, if asked.
  5. Track your Jira ticket for Completion of VDI process
  6. New joiner will receive VDI credentials in mail, after the VDI get created.

2.Request to Jump Server Access.

Process for PaaS environment jump server access –

  1. Open ARC tool through below link. https://collaborate.vodafone.com/sites/GO_User_Access_Management/SitePages/Home.aspx
  2. Search word ‘Security Group Membership’ and select
  3. Click on Security Group Membership (NON administrative)
  4. Ans select as per below:
  5. Start request
  6. Security group name : FSA-Remote_JUMP_OPENSTACK_TSS-R
  7. Owner details santosh.karpe@vodafone.com
  8. User name:type mail id of requester
  9. Submit the form.
  10. Wait the Jump Server Access request get approved. Connect with your Line Manager or Team members to get this process done asap.

3. Connect to RDP using Jump Server.

  1. New joiner have received Jump server pdf in the mail from PaaS team. Follow the PDF to connect with RDP. Use the credentials provided in VDI creation process
  2. Never shut down the PDP, always disconnect or restart it.

4. Get access to the repository.

  1. Check for the access into the repository. If access is not provided, connect with your team and ask to add you into the group.
  2. Once access is provided, Clone the repository in your RDP.

5. Install the required Software for Development.

  1. Install the required softwares in the RDP.
  2. All development should be done inside the RDP.

8. VPN setup along with MFA (Multi Factor Authentication)
